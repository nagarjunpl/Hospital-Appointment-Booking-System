# 🏥 Hospital Management System (MERN Stack)

A Hospital Management System built using the MERN Stack that helps manage hospital operations such as appointments, doctor management, billing, and online payments. The system provides separate interfaces for Patients, Doctors, and Admins.

---

## 🚀 Features

### 👤 Patient
- Secure registration and login
- View doctor profiles and specialties
- Book doctor appointments
- Book ambulance service
- View appointment history
- View invoices and make payments
- Online payment using Khalti

### 👨‍⚕️ Doctor
- Manage doctor profile
- View patient appointment requests
- Accept or reject appointments
- Manage patient prescriptions

### 🛠️ Admin
- Secure admin login
- Add, view, and delete doctors
- View all registered patients
- Manage appointments
- Generate invoices
- Track payments and billing details

---

## 🔐 Security
- Duplicate email registration is not allowed
- Passwords are encrypted
- Password fields are hidden in the admin panel
- Role-based access control
